TokenLoan
===================

A token to token loaning system based on block chain

Requirements
-------------
* Node v5+
* [Grunt-cli](http://gruntjs.com/getting-started#installing-the-cli)
* [bower](https://bower.io/#install-bower)

Install
-------------
```
npm install -g bower
npm install -g grunt-cli
npm install
bower install
grunt
```

Test
-------------
```
npm test
```
