Install
-------------
```
git clone https://github.com/tokenloan/tk_main_public.git
cd tk_main_public
npm install
```

Test
-------------
### Run contract tests:
```
npm test
```
### Run interface tests:
```
npm run test-dapp
```

Deploy
-------------
### Deploy multisig wallet:
```
truffle migrate 
```
